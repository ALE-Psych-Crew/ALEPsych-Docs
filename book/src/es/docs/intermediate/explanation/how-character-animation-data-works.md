# Como Funcionan los Datos de Animacion de Personajes

::: callout warning "Sin terminar (WIP)"
Esta pagina todavia se esta trabajando y puede cambiar.
:::

La configuracion del personaje mapea nombres de animacion y datos de sprites a acciones de gameplay.

- Los aciertos de notas llaman animaciones de sing.
- Los fallos llaman animaciones de miss.
- El icono del HUD y el color de la barra de vida provienen de campos de la configuracion del personaje.
