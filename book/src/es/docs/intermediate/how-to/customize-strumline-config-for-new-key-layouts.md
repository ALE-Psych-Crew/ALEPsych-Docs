# Personalizar la configuración de StrumLine para nuevas distribuciones de teclas

::: callout warning "Incompleto (WIP)"
Esta página todavía se está elaborando y puede cambiar.
:::

## Objetivo

Cambiar el comportamiento de la strumline y las asignaciones para distintas configuraciones de carriles.

## Pasos

1. Crea/edita `data/strumLines/<id>.json`.
2. Configura entradas `config[]` con ids/grupos de key binds.
3. Mapea cada carril a nombres de nota/splash/animación.
4. Asegúrate de que existan los IDs de strums/notes/splashes referenciados.
5. Prueba la entrada por carril y el comportamiento de acierto/fallo.
