## Meta

Configure el mapeo de carriles y las configuraciones visuales del juego vinculado.

## Campos principales

- `spacing`
- `strums`
- `notes`
- `splashes`
- `config[]`

`config[]` controla las combinaciones de teclas, los identificadores de notas, los identificadores de presentación y las asignaciones de animaciones de canto/fallo.
