# Usar el esquema JSON de Character

::: callout warning "Incompleto (WIP)"
Esta página todavía se está elaborando y puede cambiar.
:::

## Objetivo

Construir JSON de personaje con campos que el runtime realmente usa.

## Campos centrales requeridos

- `images`
- `type`
- `animations`
- `icon`
- `cameraOffset`
- `barColor`

## Pasos

1. Comienza desde un JSON de personaje conocido que funcione.
2. Mantén consistentes los nombres de animación de notas.
3. Valida las rutas de imágenes y el tipo de atlas.
