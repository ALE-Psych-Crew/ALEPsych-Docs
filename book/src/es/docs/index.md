# Bienvenido a la documentación de ALE Psych

![ALE Psych Logo](https://github.com/ALE-Psych-Crew/ALE-Psych/raw/main/art/githubLogo.png)

*"Si Psych era tan bueno, ¿por qué no hubo Psych 2?"*

Esta es la documentación oficial para el **motor ALE Psych**.

ALE Psych es un motor Friday Night Funkin' inspirado en Psych Engine, centrado en codificación suave, flujos de trabajo JSON y personalización basada en scripts.

::: callout info
Este motor es un trabajo en progreso, pero ya se utiliza para crear modificaciones completas.
:::

## Empieza aquí

- Abre el [Tabla de contenidos](./table-of-contents)
- Ve a [Principiante](./beginner) si este es tu primer mod
- Visita [Comunidad](./community) para modificaciones y marcos.

- Configuración del mod y estructura de carpetas.
- Flujos de trabajo de canciones, semanas, personajes y escenarios.
- Personalización de HUD, notas, rasgueos y salpicaduras.
- Solución de problemas comunes de modificación
