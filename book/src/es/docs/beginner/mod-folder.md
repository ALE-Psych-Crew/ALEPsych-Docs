Creo que se explica sola, pero por si acaso :blobCookie:

1. Abre `ROOT/mods/`
2. Crea una carpeta con el nombre de tu *mod*
3. Ese nombre aparecerá en el selector de mods

::: callout tip Pro Tip :gentleBlob:
Puedes abrir el menú de selección de *mods* con `Ctrl + Shift + M`
-# Esto puede variar si cambiaste tus controles
:::

## Nombres reservados (no uses esto como nombre de mod)

El selector ignora carpetas con estos nombres:

- `characters`
- `events`
- `noteTypes`
- `data`
- `songs`
- `music`
- `sounds`
- `shaders`
- `videos`
- `images`
- `stages`
- `weeks`
- `fonts`
- `scripts`
