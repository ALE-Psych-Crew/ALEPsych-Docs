# Carpeta del Mod

## Nombres reservados (no uses esto como nombre de mod)

El selector ignora carpetas con estos nombres:

- `characters`
- `events`
- `noteTypes`
- `data`
- `songs`
- `music`
- `sounds`
- `shaders`
- `videos`
- `images`
- `stages`
- `weeks`
- `fonts`
- `scripts`
