# Carpeta del Mod

Esta es la base de todo tu proyecto :blobCookie:

## Crear el mod

1. Abre `ROOT/mods/`.
2. Crea una carpeta con el nombre de tu mod.
3. Ese nombre aparecerá en el selector de mods del juego.

Ejemplo:

```text
ROOT/
  mods/
    MiPrimerMod/
```

::: callout tip Pro Tip :gentleBlob:
Puedes abrir el menú de selección de mods con `Ctrl + Shift + M`.
<sub>Este atajo puede cambiar según tu configuración de controles.</sub>
:::

## Regla importante del nombre de carpeta

No uses estos nombres reservados como nombre de tu mod, porque el motor los ignora en el selector:

- `characters`
- `events`
- `noteTypes`
- `data`
- `songs`
- `music`
- `sounds`
- `shaders`
- `videos`
- `images`
- `stages`
- `weeks`
- `fonts`
- `scripts`
