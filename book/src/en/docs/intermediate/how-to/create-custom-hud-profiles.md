# Create Custom HUD Profiles

::: callout warning "Unfinished (WIP)"
This page is still being worked on and may change.
:::

## Goal

Create a HUD profile and attach it to a stage.

## Steps

1. Create `data/huds/<id>.json`.
2. Set combo/countdown/antialiasing values.
3. Reference that HUD id from stage JSON (`hud`).
4. Add matching HUD art assets (`images/hud/<id>/...`).
5. Test countdown, combo popups, and bar visuals.
