# Add Custom Icons and Healthbar Color Behavior

::: callout warning "Unfinished (WIP)"
This page is still being worked on and may change.
:::

## Goal

Make character icons and health bar colors match your characters.

## Steps

1. Set character `icon` and `barColor` in character JSON.
2. Add matching icon data/image if custom icon id is used.
3. Test icon swap and bar color in gameplay HUD.

## Verification

- Opponent and player icons update when characters load/change.
- Health bar side colors follow character `barColor`.
