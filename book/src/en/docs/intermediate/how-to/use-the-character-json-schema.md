# Use the Character JSON Schema

::: callout warning "Unfinished (WIP)"
This page is still being worked on and may change.
:::

## Goal

Build character JSON with fields runtime actually uses.

## Required core fields

- `images`
- `type`
- `animations`
- `icon`
- `cameraOffset`
- `barColor`

## Steps

1. Start from a known working character JSON.
2. Keep note animation names consistent.
3. Validate image paths and atlas type.
