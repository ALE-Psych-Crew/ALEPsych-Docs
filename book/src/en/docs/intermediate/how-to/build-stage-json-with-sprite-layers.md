# Build Stage JSON with Sprite Layers

::: callout warning "Unfinished (WIP)"
This page is still being worked on and may change.
:::

## Goal

Use `spritesConfig` to control stage object layering.

## Steps

1. Define base stage settings (`zoom`, `speed`, `hud`).
2. Add `spritesConfig.directory` for stage image folder.
3. Add ordered `sprites` entries.
4. Use `addMethod`/properties to control placement.
5. Test rendering order in-game.
