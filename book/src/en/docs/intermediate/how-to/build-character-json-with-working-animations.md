# Build Character JSON with Working Animations

::: callout warning "Unfinished (WIP)"
This page is still being worked on and may change.
:::

## Goal

Create a character JSON that animates correctly during gameplay.

## Steps

1. Start from a known working character JSON.
2. Set `images`, `type`, and animation entries.
3. Ensure note-related animation names exist.
4. Set `icon`, `barColor`, offsets.
5. Test sing, miss, idle, and optional special animations.

## Example keys to check

- `animations[].name`
- `animations[].prefix`
- `animations[].frameRate`
- `icon`
- `barColor`
