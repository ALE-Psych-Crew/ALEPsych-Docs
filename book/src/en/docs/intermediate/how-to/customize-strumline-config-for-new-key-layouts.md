# Customize StrumLine Config for New Key Layouts

::: callout warning "Unfinished (WIP)"
This page is still being worked on and may change.
:::

## Goal

Change strumline behavior and bindings for different lane setups.

## Steps

1. Create/edit `data/strumLines/<id>.json`.
2. Set `config[]` entries with key bind ids/groups.
3. Map each lane to note/splash/animation names.
4. Ensure referenced strums/notes/splashes IDs exist.
5. Test lane input and hit/miss behavior.
