# Welcome to ALE Psych Documentation

![ALE Psych Logo](https://github.com/ALE-Psych-Crew/ALE-Psych/raw/main/art/githubLogo.png)

*"If Psych was so good, why wasn't there a Psych 2?"*

This is the official documentation for the **ALE Psych engine**.

ALE Psych is a Friday Night Funkin' engine inspired by Psych Engine, focused on softcoding, JSON workflows, and scripting-based customization.

::: callout info
This engine is a work in progress, but it is already used to build complete mods.
:::

## Start here

- Open the [Table of Contents](./table-of-contents)
- Go to [Beginner](./beginner) if this is your first mod
- Visit [Community](./community) for mods and frameworks

## What this docs site covers

- Mod setup and folder structure
- Song, week, character, and stage workflows
- HUD, notes, strums, and splashes customization
- Troubleshooting common modding issues
