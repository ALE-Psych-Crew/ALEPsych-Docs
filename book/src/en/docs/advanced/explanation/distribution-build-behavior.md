# Distribution Build Behavior

::: callout warning "Unfinished (WIP)"
This page is still being worked on and may change.
:::

Distribution behavior is mainly controlled by mod selection mode and startup path resolution.

- Forced mode uses `CONTENT_MOD` and sets `UNIQUE_MOD`.
- Standard mode resolves from CLI/save.

Practical effect: distribution packs can be locked to one content mod.
