## Mod Structure

Although the *Engine* is based on *Psych*, the structure is completely different, with the goal of reducing the size and clutter of `MOD/`

::: collapsible open `MOD/`
    ::: collapsible `data/`
        - `characters/`
        - `huds/`
        - `notes/`
        - `splashes/`
        - `stages/`
        - `strumLines/`
        - `strums/`
        - `weeks/`
    :::

    ::: collapsible `scripts/`
        - `classes/`
        - `states/`
        - `substate/`
    :::

    ::: collapsible `songs/`
        ::: collapsible `SONG/`
            - `charts/`
            - `audios/`
            - `scripts/`
        :::
    :::

    - `fonts/`
    - `images/`
    - `music/`

    - `shaders/`
    - `sounds/`
:::

::: callout tip Pro Tip :gentleBlob:
If the folders for some reason do not work
Whether due to a documentation typo or because the page is outdated

You can use the `ROOT/assets` folder as a base
:::
