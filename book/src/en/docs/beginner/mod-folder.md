## Create your mod

1. Open `ROOT/mods/`.
2. Create a folder with your mod name.
3. That folder name appears in the in-game mod selector.

Example:

```text
ROOT/
  mods/
    MyFirstMod/
```

::: callout tip Pro Tip :gentleBlob:
You can open the mod selection menu with `Ctrl + Shift + M`.
<sub>This shortcut may differ if you changed controls.</sub>
:::

## Important folder-name rule

Do not use these reserved names as your mod folder name, because the selector ignores them:

- `characters`
- `events`
- `noteTypes`
- `data`
- `songs`
- `music`
- `sounds`
- `shaders`
- `videos`
- `images`
- `stages`
- `weeks`
- `fonts`
- `scripts`
