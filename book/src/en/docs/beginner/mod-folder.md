# Mod Folder

## Reserved names (do not use this as your mod name)

The selector ignores folders with these names:

- `characters`
- `events`
- `noteTypes`
- `data`
- `songs`
- `music`
- `sounds`
- `shaders`
- `videos`
- `images`
- `stages`
- `weeks`
- `fonts`
- `scripts`
