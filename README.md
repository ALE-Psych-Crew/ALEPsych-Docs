# ALEPsych Docs

- This repository contains the ALEPsych documentation site built with **mdBook**.
- Source content lives in `book/src`.
- Run locally: `mdbook serve book`
- Build static site: `mdbook build book`
- GitHub Pages deployment is handled by the Actions workflow at `.github/workflows/mdbook-pages.yml`.
- One-time repo setting: **Settings → Pages → Source = GitHub Actions**.
